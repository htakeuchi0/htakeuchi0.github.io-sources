# 行列積の計算時間

## 概要

行列の積は2重のfor文で計算できますが，プログラムでの行列の表現方法にはいくつかの選択肢があります．
本節では，行列の表現方法や，行列積の計算手順などによって，実行時間にどのような差がでるかを実験的に確認します．
